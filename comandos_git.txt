git init .               -> cria um repo git na pasta atual 
git status               -> mostra status do trabalho (de cada arquivo)
git add "nome arquivo "  -> adiciona arquivo para ser commitado
git commit -m "nme do slv"-> faz commit do que estava para ser commitado
git log                    -> mostra todo o histórico de commit
